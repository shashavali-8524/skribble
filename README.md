# Skribble Clone

A multiplayer drawing and guessing game clone.

## Description

This project is a web-based multiplayer game where players take turns drawing words while others try to guess them.

## Features

- Real-time multiplayer gameplay
- Drawing tools (brush size, color)
- Chat functionality for guessing and communication
- Room management

## Tech Stack

- Node.js
- Socket.io
- HTML/CSS/JavaScript
